Tecnologias Utilizadas

Seja bem vindo à pagina de desenvolvimento do site, nesse local será feito as alterações de produtos e a verificação das informações dos clientes que adiquiriam nosso produto.

Completando com as tecnologias que foram utilizadas pra o desenvolvimento de toda essa página:

Front-end: Utilizado o framework Angular,bootstrap e Nodejs juntamente com as linguagens TypeScript e JavaScript, o bootstrap é um framework que ajuda bastante em um belo design, oferecendo diversos componentes e com um ótimo estilo de site responsivo, utilizado a biblioteca NgxMaskPage para utilizar mascaras nos inputs dos formulários; O NodeJs sem dúvida um dos melhores frameworks já visto, trazendo a possibilidade de rodar JavaScript e TypeScript fora do navegador-web,

Back-end No Back-end foi utilizado a linguagem Java, utilizando as bibliotecas do framework SpringBot, fazendo uso do Hibernate que é um framework ORM junto com a interface JPA para a persistência de dados na Database e o framework Lombok para utilizar getters, setters, constructor e entre outros para criação de objetos, o banco de dados utilizado foi o MySQL;
